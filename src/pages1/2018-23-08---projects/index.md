## PSAlphaFS


The AlphaFS library overcomes the MAX_PATH limitation of 260 characters and PSAlphaFS is a PowerShell wrapper providing powerShell functions that attempt to mimic the functionality of those shipped by microsoft.

[View Github][PSAlphaFS] </br>
[View PSGallery][PSAlphaFS-Gallery]

## PSLiteDB


PSLiteDB is a PowerShell wrapper for LiteDB which is a noSQL singlefile datastore just like SQLite. PSLiteDB has been compiled against the .NET Standard 2 which means you can use this module with both Windows PowerShell and PowerShell Core.

[View Github][PSLiteDB] </br>
[View PSGallery][PSLiteDB-Gallery]

## PSSecret


PSSecret acts as your personal secure vault where you can encrypt, store and retrive data securely. You can store the following types of information :
Strings,Credentials,Hashtables,Objects
The values are encrypted and stored in the current user's registry. Information is encrypted using PKI.

[View Github][PSSecret] </br>
[View PSGallery][PSSecret-Gallery]

## PowerShell Library for Excel

PowerShell wrapper for spreadsheetlight that allows you to Create, Modify multiple excel files simultaneouly without having MS Office installed. Create documents using builtin-styles,themes, or build your own.
